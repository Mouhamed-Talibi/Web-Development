Php Composer : 
--------------

    * Composer Is The Dependency Manager of Php.
    * Dependency Means : What Your Project Based On To Work Effectively.
    * Composer Is Like Npm, Pip.
    * Composer Manage Your Packages.
    * Use Cases of Composer :
        - Install A Package
        - Update A Package
        - Remove A Package

    - How To Downlaod Composer : 
        * https://getcomposer.org/
